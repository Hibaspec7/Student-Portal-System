# Student Portal System

A simple student portal system example built with HTML, CSS, and JavaScript.

## Features

- Student login form with a sample credential
- Dashboard view with courses, recent grades, and announcements
- Responsive layout using CSS grid
- Clean styling for a portal-style user interface

## Files

- `index.html` — Contains the student portal UI and the JavaScript login/dashboard logic
- `style.css` — Provides styling for the portal, forms, cards, and responsive layout
- `README.md` — Project overview and usage instructions

## How to use

1. Open `index.html` in your browser.
2. Use the sample login credential:
   - Student ID: `S123456`
   - Password: `password123`
3. After signing in, the dashboard displays:
   - current courses
   - recent grades
   - announcements

## Notes

This is a static prototype, so the login flow and data are built into the page for demonstration purposes.

## Customize

To extend this portal system, you can:

- add real authentication with a backend service
- connect course and grade data from an API or database
- add profile management, messaging, and schedule features
- improve accessibility and form validation
